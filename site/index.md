## 安装

----

- 基于 GFM 的 markdown 语法编写
- 支持 html 混排，再复杂的 demo 也可以做！
- 导出网页或者 pdf 更容易分享
- 支持单页背景图片
  - 多种模式：纵览模式，双屏模式，远程控制
  - 可以使用画板，可以使用 note 做备注
  - 支持语法高亮，自由选择 highlight 样式
  - 可以单页 ppt 内部动效，单步动效
